# ビジュアルコンセプト統合定義書
## クリエイター向けハイエンド電動自転車

### プロジェクト概要
本ドキュメントは、ムードボードコンセプトと表現モチーフ定義書を統合し、実装可能な統一ビジュアルコンセプトを策定したものです。全てのタッチポイントで一貫したブランド体験を提供するための包括的ビジュアル戦略を定義します。

---

## 1. 統合ビジュアルコンセプト（コアアイデア）

### 1.1 コンセプトステートメント

**"Urban Crafted Intelligence"**
**都市を走る、創造的知性**

都市という舞台で活動するクリエイターのために設計された、知性と感性が融合したモビリティツール。洗練されたミニマリズムの中に職人の手仕事を宿し、先進テクノロジーを静かに纏う。それは単なる移動手段ではなく、創造的ライフスタイルを支え、インスピレーションを生み出すパートナー。

### 1.2 ビジュアルコンセプトの 4 つの柱

#### Pillar 1: Minimalist Precision（ミニマリスト精密性）
**本質**: Less is More の徹底追求

- **視覚表現**:
  - 削ぎ落とされたフォルム
  - 幾何学的正確性
  - ネガティブスペースの戦略的活用
  - クリーンなラインと鋭角的エッジ

- **実装原則**:
  - 70% をモノクロームで構成
  - ネガティブスペース 40-60%
  - グリッドシステム（8px ベース）厳守
  - 装飾の完全排除

#### Pillar 2: Artisan Detail（職人的ディテール）
**本質**: 手仕事の温もりと誇り

- **視覚表現**:
  - レザーの経年変化
  - 真鍮の鈍い輝き
  - 木目の自然なパターン
  - ステッチやリベットの視覚化

- **実装原則**:
  - マクロ撮影でディテール強調
  - 素材感を触覚的に伝える質感表現
  - エイジングプロセスの可視化
  - クラフツマンシップストーリーの統合

#### Pillar 3: Tech-Forward Elegance（先進的エレガンス）
**本質**: 未来を静かに纏う

- **視覚表現**:
  - シームレスなデジタルインターフェース
  - LED による機能的美
  - カーボンファイバーの織り目
  - 空力デザインの流線形

- **実装原則**:
  - テクノロジーを主張せず統合
  - Electric Blue をテックアクセントに
  - UI/UX の直感的ビジュアル化
  - 機能美の徹底追求

#### Pillar 4: Sustainable Integrity（持続可能な誠実さ）
**本質**: 環境への責任を美しく

- **視覚表現**:
  - リサイクル素材の質感
  - 自然との調和
  - 製造プロセスの透明性
  - 長寿命設計の証明

- **実装原則**:
  - Forest Green でエコ認証表現
  - モジュラー設計の可視化
  - サステナビリティストーリーの統合
  - 環境データの美しいビジュアライゼーション

### 1.3 ビジュアルコンセプトマップ

```
                Urban Crafted Intelligence
                          |
        +-----------------+-----------------+
        |                 |                 |
   Minimalist       Artisan Detail    Tech-Forward
   Precision                            Elegance
        |                 |                 |
        +--------+--------+--------+--------+
                 |                 |
            Sustainable       Creative
             Integrity        Empowerment
```

**統合原理**: 相反する要素の調和
- 冷たい金属 × 温かいレザー
- 幾何学的 × 有機的
- デジタル × アナログ
- グローバル × ローカル

## 2. ビジュアル言語の定義

### 2.1 カラー言語システム

#### Primary Visual Language（プライマリー視覚言語）

**Monochrome Authority（モノクローム権威）**
- **Deep Charcoal** (#2C2C2E): 力強さ、信頼、都市の夜
- **Titanium Grey** (#8E8E93): 精密さ、テクノロジー、中立性
- **Pure White** (#FFFFFF): 純粋さ、開放感、可能性
- **Matte Black** (#1C1C1E): 高級感、深み、存在感

**使用比率**: 全体の 70%
**用途**: ブランドアイデンティティの基盤、製品本体、大面積

#### Secondary Visual Language（セカンダリー視覚言語）

**Natural Craft Voice（ナチュラルクラフトボイス）**
- **Cognac Brown** (#8B5A3C): 職人技、温かみ、時間の美
- **Brushed Copper** (#B87333): 伝統、耐久性、手仕事
- **Warm Oak** (#C19A6B): 自然、有機性、持続可能性

**使用比率**: 全体の 20%
**用途**: アクセントパーツ、プレミアムエディション、ディテール

**Tech Highlight Voice（テックハイライトボイス）**
- **Electric Blue** (#007AFF): イノベーション、接続性、デジタル
- **Neon Lime** (#32D74B): エネルギー、アクティブ、充電完了
- **Amber Alert** (#FF9500): 注意、安全、視認性

**使用比率**: 全体の 10%
**用途**: UI、インジケーター、インタラクティブ要素

#### Color Communication Rules（カラーコミュニケーションルール）

1. **70-20-10 ルールの厳守**
   - 基盤 70%: モノクローム
   - 調和 20%: ナチュラルクラフト
   - アクセント 10%: テックハイライト

2. **色の意味の一貫性**
   - Electric Blue = デジタル、接続、イノベーション（常に）
   - Neon Lime = 準備完了、ポジティブアクション（常に）
   - Amber = 警告、注意喚起、安全（常に）

3. **コンテキストによる調整**
   - Web: sRGB、鮮やかさ維持
   - 印刷: CMYK 変換、彩度 -10%
   - SNS: 高コントラスト、視認性優先
   - 動画: HDR 対応、色温度統一

### 2.2 タイポグラフィ言語システム

#### Typographic Voice（タイポグラフィックボイス）

**Precision Voice（精密ボイス）**
- **フォント**: Inter Variable (Extra Light - Black)
- **特徴**: 幾何学的、ニュートラル、高可読性
- **用途**: ヘッドライン、ブランドメッセージ、キャンペーン
- **声のトーン**: 自信、明晰、モダン

**Human Voice（人間ボイス）**
- **フォント**: SF Pro Text / Noto Sans Japanese
- **特徴**: 親しみやすい、読みやすい、普遍的
- **用途**: 本文、説明文、対話的コンテンツ
- **声のトーン**: 親切、誠実、理解しやすい

**Technical Voice（テクニカルボイス）**
- **フォント**: JetBrains Mono
- **特徴**: 等幅、正確、数値の視認性
- **用途**: スペック、データ、技術情報
- **声のトーン**: 客観的、信頼できる、詳細

**Artisan Voice（職人ボイス）**
- **フォント**: Playfair Display
- **特徴**: セリフ、エレガント、伝統的
- **用途**: ストーリーテリング、引用、エディトリアル
- **声のトーン**: 情熱的、物語性、時間を超越

#### Typography Hierarchy（タイポグラフィ階層）

**Level 1: Hero Statement**
- **サイズ**: 64pt (デスクトップ) / 40pt (モバイル)
- **フォント**: Inter Extra Light
- **用途**: ランディングページ第一印象
- **効果**: 静かな存在感、圧倒的余白

**Level 2: Section Anchor**
- **サイズ**: 48pt (デスクトップ) / 32pt (モバイル)
- **フォント**: Inter Regular
- **用途**: セクション導入
- **効果**: 構造の明確化、視線誘導

**Level 3: Content Leader**
- **サイズ**: 32pt (デスクトップ) / 24pt (モバイル)
- **フォント**: Inter Semi Bold
- **用途**: サブセクション見出し
- **効果**: 情報階層の明示

**Level 4: Body Text**
- **サイズ**: 16pt (デスクトップ) / 14pt (モバイル)
- **フォント**: SF Pro Text Regular / Noto Sans JP
- **用途**: 本文、説明
- **効果**: 可読性、理解促進

**Level 5: Support Text**
- **サイズ**: 12pt
- **フォント**: SF Pro Text Regular
- **用途**: キャプション、注釈
- **効果**: 補足情報提供

### 2.3 形状言語システム

#### Geometric Vocabulary（幾何学的ボキャブラリー）

**The Circle: 完全性と連続性**
- **意味**: 循環、エンドレスジャーニー、ゼロカーボン
- **使用**: ホイール、ボタン、ローディング、ロゴエレメント
- **サイズ系統**: 8px, 24px, 48px, 96px, 192px
- **視覚効果**: 柔らかさ、包括性、調和

**The Line: 方向性と秩序**
- **意味**: 道、進路、ミニマリズム、構造
- **使用**: フレーム、セパレーター、アンダーライン、地平線
- **太さ系統**: 1px (繊細), 2-3px (標準), 4-8px (強調)
- **視覚効果**: クリーンさ、方向性、秩序

**The Triangle: 安定と進歩**
- **意味**: フレーム構造、前進、強度、バランス
- **使用**: ナビゲーション、アイコン、構図、力学表現
- **角度系統**: 60° (正三角), 45° (直角二等辺), 90° (直角)
- **視覚効果**: 動的、安定、力強い

**The Rectangle: 構造と安定**
- **意味**: 建築性、フレーム、基盤、信頼性
- **使用**: レイアウト、UI カード、フレーム、グリッド
- **比率系統**: 16:9 (ワイド), 4:3 (標準), 1:1.618 (黄金比)
- **視覚効果**: 安定、構造的、信頼できる

#### Organic Vocabulary（有機的ボキャブラリー）

**Flowing Curves: 動きと快適性**
- **意味**: 人間工学、流動性、自然な動き
- **使用**: サドル、グリップ、ハンドルバー曲線
- **カーブ率**: ベジェ 0.55 ハンドル比
- **視覚効果**: 柔軟、快適、人間中心

**Ergonomic Contours: 人との調和**
- **意味**: フィット感、快適性、身体性
- **使用**: 接触面、グリップ、サドル形状
- **基準**: ISO 5356 人体測定データ
- **視覚効果**: 親和性、使いやすさ、配慮

### 2.4 質感言語システム

#### Tactile Vocabulary（触覚的ボキャブラリー）

**Matte Silence（マットな静寂）**
- **素材**: アルマイトアルミ、粉体塗装
- **反射率**: 5-15%
- **感覚**: ソフト、上品、指紋レス
- **意味**: 洗練、品質、モダニティ

**Brushed Precision（ブラッシュド精密）**
- **素材**: ヘアライン仕上げステンレス
- **方向性**: 一方向研磨パターン
- **感覚**: 方向性、産業美、精密
- **意味**: テクノロジー、製造の誇り

**Organic Warmth（有機的温かみ）**
- **素材**: 植物タンニンなめしレザー
- **経年変化**: 艶と深みの増加
- **感覚**: 温かい、生きている、唯一無二
- **意味**: クラフツマンシップ、時間の価値

**Carbon Weave（カーボンウィーブ）**
- **素材**: カーボンファイバー綾織り
- **パターン**: 2x2 Twill 可視化
- **感覚**: ハイテク、軽量、強度
- **意味**: イノベーション、先進性、効率

**Natural Grain（ナチュラルグレイン）**
- **素材**: オイル仕上げアッシュ材
- **パターン**: 木目、年輪
- **感覚**: 自然、有機的、生命力
- **意味**: サステナビリティ、自然回帰

## 3. 主要ビジュアル要素の使用ガイドライン

### 3.1 プロダクトビジュアルガイドライン

#### Product Photography Standards（プロダクト撮影基準）

**Studio Shot（スタジオショット）**

**セットアップ**:
- 背景: Pure White seamless (#FFFFFF)
- ライティング: 3 点照明（キー、フィル、リム）
- 色温度: 5500K（デイライト）
- カメラ: フルフレーム、50mm または 85mm
- 絞り: f/8 - f/11（全体シャープ）

**アングル構成**:
1. **Hero Angle（ヒーローアングル）**: 斜め 45°、目線やや下
   - 用途: メインビジュアル、ランディングページ
   - 効果: 力強さ、存在感、全体像

2. **Profile View（プロファイルビュー）**: 真横、水平
   - 用途: プロポーション表現、スペック説明
   - 効果: 正確性、フォルム理解

3. **Top-Down View（トップダウンビュー）**: 真上、俯瞰
   - 用途: 構造理解、レイアウト説明
   - 効果: 全体性、コンポーネント関係

4. **Detail Shot（ディテールショット）**: マクロ、部分
   - 用途: 素材感、クラフツマンシップ表現
   - 効果: 品質証明、触覚的訴求

**ライティングパターン**:
- **High Key（ハイキー）**: 明るい、軽快、モダン
  - 使用: 標準モデル、明るい印象
- **Low Key（ローキー）**: 暗い、ドラマティック、高級
  - 使用: プレミアムモデル、感情的訴求
- **Rembrandt（レンブラント）**: 三角光、立体的
  - 使用: ポートレート風、ストーリー性

**Environmental Context Shot（環境コンテキストショット）**

**Urban Morning（アーバンモーニング）**
- **時間帯**: 日の出後 1 時間（ゴールデンアワー）
- **場所**: ガラス張りビル、石畳、橋
- **被写体**: 自転車 + 環境のバランス（40:60）
- **効果**: インスピレーション、始まりの予感

**Creative Studio（クリエイティブスタジオ）**
- **環境**: ロフト、アトリエ、デザイン事務所
- **構成**: 自転車 + 創作ツール（カメラ、MacBook など）
- **人物**: ぼかし、シルエット、部分（手のみ等）
- **効果**: ライフスタイル訴求、実用性証明

**Night Tech（ナイトテック）**
- **時間帯**: 夜間（ブルーアワー）
- **照明**: LED ライト強調、都市光
- **環境**: モダン都市、ネオン、ガラス反射
- **効果**: 先進性、テクノロジー、都会的

#### Visual Composition Rules（視覚構成ルール）

**Rule of Thirds（三分割法）**
- グリッド: 縦横 3 分割
- 配置: 主要素を交点に
- 用途: 安定感、自然な視線誘導

**Golden Ratio（黄金比）**
- 比率: 1:1.618
- 適用: フレーム比率、要素配置
- 効果: 美的調和、自然な美しさ

**Negative Space Dominance（ネガティブスペース優位）**
- 比率: 被写体 40% / 余白 60%
- 効果: 高級感、余裕、ミニマリズム
- 用途: ヒーローイメージ、ブランドページ

**Leading Lines（リーディングライン）**
- 要素: 道路、建物のライン、フレーム
- 効果: 視線誘導、奥行き、動的
- 用途: アクションショット、ストーリーテリング

### 3.2 UI/UX ビジュアルガイドライン

#### Interface Visual Language（インターフェース視覚言語）

**Component Library（コンポーネントライブラリー）**

**Buttons（ボタン）**
- **Primary Button**:
  - 色: Deep Charcoal (#2C2C2E)
  - 形: 角丸 8px
  - 高さ: 48px（デスクトップ）、56px（モバイル）
  - ホバー: Titanium Grey + 2px 上昇
  - アニメーション: 200ms ease-out

- **Secondary Button**:
  - 色: Titanium Grey (#8E8E93) ボーダー + 透明背景
  - 形: 角丸 8px
  - 高さ: 48px（デスクトップ）、56px（モバイル）
  - ホバー: 背景塗りつぶし
  - アニメーション: 200ms ease-out

- **CTA Button**:
  - 色: Electric Blue (#007AFF)
  - 形: 角丸 8px
  - 高さ: 56px（強調）
  - ホバー: 輝度 +10% + 4px 上昇
  - アニメーション: 300ms cubic-bezier(0.65, 0, 0.35, 1)

**Cards（カード）**
- **背景**: Pure White (#FFFFFF)
- **影**: 0 4px 16px rgba(0,0,0,0.08)
- **角丸**: 16px
- **ホバー**: 0 8px 24px rgba(0,0,0,0.12) + 4px 上昇
- **内側余白**: 32px（デスクトップ）、24px（モバイル）

**Forms（フォーム）**
- **Input Field**:
  - 高さ: 48px
  - ボーダー: 1px solid Titanium Grey
  - 角丸: 8px
  - フォーカス: Electric Blue ボーダー 2px
  - 背景: Pure White

- **Label**:
  - フォント: SF Pro Text Regular 14pt
  - 色: Matte Black (#1C1C1E)
  - 位置: Input 上部 8px

- **Error State**:
  - ボーダー: Amber Alert (#FF9500) 2px
  - アイコン: 警告三角形
  - メッセージ: Amber Alert 色

**Navigation（ナビゲーション）**
- **Header**:
  - 高さ: 72px（デスクトップ）、64px（モバイル）
  - 背景: Pure White + blur
  - 影: 0 2px 8px rgba(0,0,0,0.04)
  - スティッキー: スクロール時固定

- **Menu Items**:
  - フォント: Inter Regular 16pt
  - 色: Deep Charcoal（通常）、Electric Blue（アクティブ）
  - ホバー: Titanium Grey 背景
  - アニメーション: 100ms ease-out

**Icons（アイコン）**
- **スタイル**: ラインアイコン、2px 線幅
- **サイズ**: 24px（標準）、48px（大）
- **色**: Titanium Grey（通常）、Electric Blue（アクティブ）
- **形状**: 円形または正方形ベース
- **一貫性**: Feather Icons または Material Icons

#### Interactive States（インタラクティブステート）

**Default（デフォルト）**
- 静止状態、最小限の視覚効果

**Hover（ホバー）**
- 色変化（微細）
- 軽微な上昇（2-4px）
- 影の深化
- 100-200ms トランジション

**Active（アクティブ）**
- 押下表現（1-2px 下降）
- 色の濃化
- 影の縮小
- 即座のフィードバック（0-50ms）

**Focus（フォーカス）**
- Electric Blue アウトライン 2px
- キーボードナビゲーション対応
- アクセシビリティ準拠

**Disabled（無効）**
- 透明度 40%
- cursor: not-allowed
- グレースケール化
- インタラクション無効

### 3.3 アイコノグラフィガイドライン

#### Icon Design Principles（アイコンデザイン原則）

**Geometric Foundation（幾何学的基礎）**
- ベース: 24x24px グリッド（2px 線幅）
- 角丸: 2px（エンドキャップ）
- 余白: 2px（グリッド端から）
- 光学調整: 円形は 1px 大きく

**Style Consistency（スタイル一貫性）**
- **線スタイル**: Round cap、Round join
- **線幅**: 2px（標準）、3px（強調）
- **塗り**: なし（ラインアイコン）
- **色**: Titanium Grey（デフォルト）

**Icon Families（アイコンファミリー）**

1. **Navigation Icons（ナビゲーションアイコン）**
   - 矢印、ホーム、メニュー、検索
   - 用途: UI ナビゲーション
   - 特徴: シンプル、即座に認識

2. **Product Feature Icons（製品機能アイコン）**
   - バッテリー、速度、距離、GPS
   - 用途: スペック表示、機能説明
   - 特徴: 技術的、正確

3. **Action Icons（アクションアイコン）**
   - ダウンロード、シェア、いいね、カート
   - 用途: ユーザーアクション
   - 特徴: 動詞的、明確

4. **Brand Icons（ブランドアイコン）**
   - 自転車シルエット、ホイール、サステナビリティマーク
   - 用途: ブランド識別、価値表現
   - 特徴: 独自性、ブランド固有

### 3.4 写真スタイルガイドライン

#### Photography Mood & Tone（写真ムードとトーン）

**Color Grading（カラーグレーディング）**
- **彩度**: -10%〜-15%（控えめ）
- **コントラスト**: +15%〜+20%（強調）
- **ハイライト**: -5%（飛ばさない）
- **シャドウ**: +10%（ディテール維持）
- **色温度**: 5500K〜6000K（クール寄り）
- **色かぶり**: Tint +2（わずかにグリーン）

**Post-Processing Signature（後処理シグネチャ）**
1. ベース露出調整
2. コントラスト S カーブ
3. 彩度控えめ調整
4. ハイライト・シャドウ保護
5. エッジシャープニング（0.5-1.0）
6. わずかなビネット（-5〜-10）
7. グレイン追加（透明度 3%、サイズ 20）

**Forbidden Styles（禁止スタイル）**
- ❌ 過度な HDR
- ❌ 強いビネット
- ❌ 派手なフィルター
- ❌ 不自然な色飽和
- ❌ ティルトシフト偽装

#### Subject Composition（被写体構成）

**Product + Human（製品 + 人間）**
- **比率**: 製品 60% / 人物 40%
- **人物表現**: 部分（手、影、シルエット）
- **焦点**: 製品シャープ、人物わずかにぼかし
- **効果**: 実用性、人間性、スケール感

**Product + Environment（製品 + 環境）**
- **比率**: 製品 40% / 環境 60%
- **環境選択**: 都市建築、自然光、ミニマル空間
- **焦点**: 製品シャープ、環境 f/2.8-4.0 ぼかし
- **効果**: コンテキスト、ライフスタイル、インスピレーション

**Product Alone（製品単独）**
- **比率**: 製品 100%
- **背景**: Pure White または環境ぼかし
- **焦点**: 全体シャープ（f/8-11）
- **効果**: 存在感、ディテール、プロフェッショナル

- ブランド方針変更
- メジャーキャンペーン
- パートナーシップビジュアル
- 承認者: CMO, CEO

#### Quality Checklist（品質チェックリスト）

**色使用チェック**
- [ ] 70-20-10 ルール遵守
- [ ] WCAG AA 以上コントラスト比
- [ ] 色のみに依存しない情報伝達
- [ ] 印刷時 CMYK 変換確認

**タイポグラフィチェック**
- [ ] 定義フォントのみ使用
- [ ] 階層構造明確
- [ ] 最小 12pt（アクセシビリティ）

- [ ] 8px グリッド整合

## 6. ブランドビジュアルのトーンとムード

### 6.1 感情的トーンマトリックス

#### Primary Emotional Tone（プライマリー感情トーン）

**Sophisticated Confidence（洗練された自信）**
- **視覚表現**: ミニマル、余白、高品質写真
- **カラー**: モノクローム優位
- **タイポグラフィ**: Inter Extra Light 大判使用
- **効果**: 静かな存在感、過度に主張しない強さ
- **適用**: ブランドページ、製品詳細、高額モデル

**Inspirational Openness（インスピレーショナルな開放感）**
- **視覚表現**: 広大な空、都市の地平線、光
- **カラー**: Sky Blue, Pure White
- **構図**: ネガティブスペース 60%
- **効果**: 可能性、自由、創造的刺激
- **適用**: キャンペーン、ランディングページ、ストーリーテリング

**Calm Focus（穏やかな集中）**
- **視覚表現**: クリーンな背景、単一被写体、ソフトライト
- **カラー**: Titanium Grey, Concrete Grey
- **ペース**: スローモーション、長いカット
- **効果**: 瞑想的、ストレスフリー、思考整理
- **適用**: 製品紹介動画、瞑想的コンテンツ

#### Secondary Emotional Tone（セカンダリー感情トーン）

**Progressive Optimism（進歩的楽観主義）**
- **視覚表現**: 朝日、新しい道、前進する人物
- **カラー**: Sunrise Orange, Neon Lime
- **動き**: 前進、上昇、展開
- **効果**: 未来志向、希望、ポジティブ変化
- **適用**: サステナビリティコンテンツ、新製品発表

**Artisan Pride（職人の誇り）**
- **視覚表現**: 手のクローズアップ、製造工程、ディテール
- **カラー**: Cognac Brown, Brushed Copper
- **質感**: レザー、木、金属
- **効果**: 手仕事への敬意、時間の価値、唯一無二性
- **適用**: クラフツマンシップストーリー、限定モデル

**Tech Elegance（テクノロジカルエレガンス）**
- **視覚表現**: LED ライト、デジタル UI、カーボン織り目
- **カラー**: Electric Blue、カーボンブラック
- **エフェクト**: グロー、パルス、データビジュアライゼーション
- **効果**: 先進性、シームレス、直感的
- **適用**: スマート機能紹介、アプリ連携

### 6.2 ムード別ビジュアルレシピ

#### Recipe 1: "Urban Morning Inspiration"（都市の朝のインスピレーション）

**使用場面**: ランディングページ、ブランドキャンペーン

**ビジュアル構成**:
- 時間帯: ゴールデンアワー（日の出後 30 分）
- 場所: ガラス張りビル、橋、広い道路
- 被写体: 自転車 + クリエイター（シルエット/部分）
- 構図: 三分割法、右下に製品

**カラーパレット**:
- 主: Sky Blue (30%), Pure White (40%)
- 副: Deep Charcoal (20%), Sunrise Orange (10%)

**タイポグラフィ**:
- ヘッドライン: Inter Extra Light 64pt、Pure White
- サブヘッド: SF Pro Text 18pt、Pure White 80%

**ムード**:
- 感情: インスピレーション、新しい始まり、可能性
- エネルギー: 穏やか、前向き、活力

**音（動画の場合）**:
- BGM: アンビエント、ピアノ、微細な都市音
- 効果音: 鳥のさえずり、タイヤの静かな回転音

#### Recipe 2: "Crafted Detail Close-up"（クラフトされたディテールクローズアップ）

**使用場面**: 製品詳細ページ、職人ストーリー

**ビジュアル構成**:
- 照明: サイドライト、リムライト強調
- 焦点: マクロ、f/2.8-4.0 浅い被写界深度
- 被写体: レザーステッチ、真鍮リベット、木目
- 構図: 中央配置、ネガティブスペース左右

**カラーパレット**:
- 主: Cognac Brown (40%), Matte Black (30%)
- 副: Brushed Copper (20%), Warm Oak (10%)

**タイポグラフィ**:
- キャプション: SF Pro Text 14pt、Matte Black
- 素材名: JetBrains Mono 12pt、Titanium Grey

**ムード**:
- 感情: 敬意、職人技への驚嘆、所有欲
- 質感: 触覚的、温かい、生きている

**音（動画の場合）**:
- BGM: アコースティックギター、チェロ
- 効果音: レザーの擦れる音、金属の微細な響き

#### Recipe 3: "Tech Night Ride"（テックナイトライド）

**使用場面**: スマート機能紹介、夜間安全性訴求

**ビジュアル構成**:
- 時間帯: ブルーアワー、夜間
- 場所: モダン都市、ネオン、ガラス反射
- 被写体: 自転車（LED ライト強調）+ 走行シーン
- 構図: ダイナミック、モーションブラー

**カラーパレット**:
- 主: Deep Charcoal (40%), Electric Blue (30%)
- 副: Neon Lime (20%), Amber Alert (10%)

**タイポグラフィ**:
- ヘッドライン: Inter Semi Bold 48pt、Electric Blue
- スペック: JetBrains Mono 18pt、Neon Lime

**ムード**:
- 感情: 先進性、安全、都会的クールさ
- エネルギー: ダイナミック、テック感、未来的

**音（動画の場合）**:
- BGM: エレクトロニカ、シンセウェーブ
- 効果音: LED 点灯音、デジタル UI 操作音

#### Recipe 4: "Sustainable Journey"（サステナブルジャーニー）

**使用場面**: 環境配慮訴求、サステナビリティレポート

**ビジュアル構成**:
- 場所: 公園、河川敷、緑地
- 被写体: 自転車 + 自然環境
- 構図: 広角、環境 60% / 製品 40%
- 季節: 春・夏（緑豊か）

**カラーパレット**:
- 主: Forest Green (30%), Pure White (30%)
- 副: Earth Brown (20%), Sky Blue (20%)

**タイポグラフィ**:
- ヘッドライン: Inter Regular 48pt、Forest Green
- データ: JetBrains Mono 24pt、Earth Brown

**ムード**:
- 感情: 責任、希望、自然との調和
- エネルギー: 穏やか、誠実、前向き

**音（動画の場合）**:
- BGM: アコースティック、フォーク
- 効果音: 風、葉のそよぎ、水の流れ

### 6.3 禁止ビジュアルスタイル

#### Forbidden Visual Styles（禁止ビジュアルスタイル）

**❌ 過度な装飾**
- 不必要なグラデーション
- 過剰なドロップシャドウ
- 複雑なパターン背景
- 多色使用（5 色以上）

**理由**: ブランドのミニマリズム哲学に反する

**❌ 低品質表現**
- ストックフォト感（過度に演出された笑顔等）
- 解像度不足
- 不自然な合成
- 過度なフィルター

**理由**: ブランドの品質基準に満たない

**❌ 競合類似表現**
- 大手自動車メーカー風
- スポーツバイク過度強調
- マスマーケット訴求
- 価格訴求中心

**理由**: ブランドの差別化ポイント不明瞭

**❌ 不適切トーン**
- 攻撃的、アグレッシブ
- 子供向け、カジュアル過ぎ
- ラグジュアリー過ぎ（超高級車風）
- 技術的過ぎ（エンジニアリング図面のみ）

**理由**: ターゲットペルソナとのミスマッチ

### 6.4 ブランドムードの一貫性測定

#### Brand Mood Score（ブランドムードスコア）

**評価項目（各 10 点満点、合計 100 点）**

1. **Minimalism（ミニマリズム）** (10pt)
   - ネガティブスペース 40% 以上: 10pt
   - 装飾要素なし: 10pt
   - 減点: 過剰な要素（-2pt/要素）

2. **Sophistication（洗練）** (10pt)
   - 高品質写真（300dpi 以上）: 5pt
   - カラーグレーディング統一: 5pt

3. **Craftsmanship（職人技）** (10pt)
   - ディテールショット含む: 5pt
   - 素材感表現: 5pt

4. **Innovation（革新性）** (10pt)
   - テック要素統合: 5pt
   - Electric Blue 効果的使用: 5pt

5. **Sustainability（持続可能性）** (10pt)
   - 環境コンテキスト: 5pt
   - サステナビリティメッセージ: 5pt

6. **Emotional Resonance（感情的共鳴）** (10pt)
   - ストーリー性: 5pt
   - ペルソナ共感性: 5pt

7. **Color Harmony（カラー調和）** (10pt)
   - 70-20-10 ルール遵守: 10pt

8. **Typographic Hierarchy（タイポグラフィ階層）** (10pt)
   - 明確な階層: 5pt
   - フォント統一: 5pt

9. **Compositional Balance（構図バランス）** (10pt)
   - 三分割法/黄金比: 5pt
   - 視線誘導明確: 5pt

10. **Brand Differentiation（ブランド差別化）** (10pt)
    - 競合との明確な違い: 5pt
    - 独自ビジュアル要素: 5pt

**スコア判定**:
- **90-100pt**: Excellent（そのまま使用可）
- **80-89pt**: Good（微調整後使用）
- **70-79pt**: Acceptable（要改善）
- **69pt 以下**: Rejected（再制作必要）

**測定頻度**:
- 新規クリエイティブ: 毎回
- 既存アセット: 四半期ごと
- キャンペーン: 実施前必須

## 結論

本ビジュアルコンセプト統合定義書は、「クリエイター向けハイエンド電動自転車」ブランドの視覚的アイデンティティを包括的に定義し、全てのタッチポイントで一貫した体験を提供するための実装可能なフレームワークです。

### コアコンセプト再確認

**"Urban Crafted Intelligence"（都市を走る、創造的知性）**

このコンセプトは、4 つの柱によって支えられています：

1. **Minimalist Precision** - Less is More の徹底追求
2. **Artisan Detail** - 手仕事の温もりと誇り
3. **Tech-Forward Elegance** - 未来を静かに纏う
4. **Sustainable Integrity** - 環境への責任を美しく

### 実装の鍵

**一貫性**が最も重要です。全てのビジュアル表現は：

- **70-20-10 カラールール**に従う
- **8px グリッドシステム**で整合する
- **明確なタイポグラフィ階層**を維持する
- **ネガティブスペース**を戦略的に活用する
- **ブランドムード**を常に反映する

### 継続的進化

このドキュメントは「生きたドキュメント」です：

- **四半期ごと**に見直し
- **フィードバック**を積極的に収集
- **新トレンド**を慎重に評価
- **ブランドコア**は不変、**表現手法**は進化

### 最終メッセージ

ビジュアルは単なる装飾ではありません。それはブランドの**思想**、**価値**、**約束**を視覚化したものです。

このガイドラインを活用し、ターゲットであるクリエイターたちに、インスピレーションと信頼を与える、一貫した美しいブランド体験を創造してください。
